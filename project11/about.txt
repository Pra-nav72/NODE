In this project, we will focus on server side rendering.
with the help of ejs templating engine.


install EJS
>>> npm i ejs

after creating .ejs file in the views folder.
** in index.js(main file)  **
-->  app.set("view engine", "ejs");

use this so that app can know the path of views files(ejs), import 'path' first.
--> app.set("views", path.resolve("./views"));
